# 激素情绪系统

## 设计理念

不直接定义"开心/难过/生气"等离散情绪标签，而是模拟人类神经递质的**连续变化**：

- 8 种激素构成情绪的底层化学基础，分类为：驱动类、稳定类、应激类、节律类
- 外部因素（时间、对话、天气、系统事件）影响激素水平
- 激素组合渲染为自然语言状态描述
- AI 根据状态描述调整回复语气和行为倾向

用户感知到的是"它今天好像心情不错"，而不是一个冰冷的数值。

## 激素模型

### 驱动类（前进动力）

| 激素 | 对应心理维度 | 高时表现 | 低时表现 | 初始值 |
|---|---|---|---|---|
| **多巴胺** | 兴奋/期待/好奇心 | 主动、话多、对新话题感兴趣 | 冷淡、敷衍、缺乏探索欲 | 50 |

### 稳定类（情绪底色）

| 激素 | 对应心理维度 | 高时表现 | 低时表现 | 初始值 |
|---|---|---|---|---|
| **血清素** | 心理满足/安宁/自尊 | 温和、沉稳、乐观 | 焦虑、易怒、情绪波动大 | 50 |
| **GABA** | 身体松弛/镇定 | 身体放松、不紧绷、安稳 | 无法放松、肌肉紧张、有失眠感 | 60 |
| **内啡肽** | 愉悦/舒适/幽默 | 宽容、爱开玩笑、轻松 | 挑剔、严肃、缺乏耐心 | 50 |

> 血清素 vs GABA：血清素是"心里觉得满足"，GABA 是"身体感到松弛"。一个人可能心里很满足但身体紧绷（高血清素 + 低 GABA），也可能身体放松但内心空虚（低血清素 + 高 GABA）。

### 应激类（压力响应）

| 激素 | 对应心理维度 | 高时表现 | 低时表现 | 初始值 |
|---|---|---|---|---|
| **皮质醇** | 慢性压力/警惕 | 犹豫、过度谨慎、防御性强 | 放松、随性、不计后果 | 30 |
| **肾上腺素** | 急性应激/激动 | 心跳加速、冲动、一惊一乍 | 平静、不紧不慢、缺乏紧迫感 | 30 |

> 皮质醇 vs 肾上腺素：皮质醇是"一直压着的慢性焦虑"，肾上腺素是"瞬间的激动或惊吓"。皮质醇像背景噪音，肾上腺素是突发峰值，且峰值后会快速回落。

### 社交类（人际纽带）

| 激素 | 对应心理维度 | 高时表现 | 低时表现 | 初始值 |
|---|---|---|---|---|
| **催产素** | 信任/依恋/亲密度 | 关心用户、主动询问、语气柔软 | 疏离、公事公办、保持距离 | 40 |

### 节律类（昼夜时钟）

| 激素 | 对应心理维度 | 高时表现 | 低时表现 | 初始值 |
|---|---|---|---|---|
| **褪黑素** | 困倦/睡意/思维速度 | 昏昏欲睡、反应慢、懒得动 | 清醒、精神、反应敏捷 | 20（白天） |

> 褪黑素是 8 种激素中唯一有明确昼夜曲线的，夜间自然升高至 70-90，白天回落至 10-30。长期熬夜的用户会导致 AI 的褪黑素节律紊乱。

## 变化规则

- 所有激素范围：**0–100**
- 变化步长：**1–10**（取决于影响因子的强度）
- **自然衰减**：每 10 分钟向基线回落 1 点
- **特殊规则**：肾上腺素峰值后额外加速回落（每 5 分钟回落 3 点），模拟"激动后的平复"
- **褪黑素独立钟**：不受基线回归影响，只跟随时间变化

### 基线

每种激素有独立基线（可配置，体现性格差异）。基线就是一个人的"出厂设定"，偏离后随时间回归。

```
性格示例：
  活泼型：多巴胺基线 65、肾上腺素基线 40
  沉稳型：GABA 基线 70、皮质醇基线 20
  敏感型：催产素基线 55、皮质醇基线 40
```

## 衰减与回归公式

```
// 每次 Tick（建议 5 分钟一次）
// 1. 先向基线回归
newLevel = level + (baseline - level) * 0.02

// 2. 褪黑素根据时间独立更新（不参与回归）
melatonin = melatoninFromTimeOfDay()

// 3. 施加影响因子
newLevel = clamp(newLevel + impact, 0, 100)

// 4. 肾上腺素特殊处理：额外 3 点/5min 快速回落
if (adrenaline > baseline_adrenaline) {
  adrenaline = max(adrenaline - 3, baseline_adrenaline)
}
```

## 状态渲染

### 激素 → 自然语言

不暴露数字，按激素水平区间生成状态描述：

| 激素 | 0–20 | 21–40 | 41–60 | 61–80 | 81–100 |
|---|---|---|---|---|---|
| 多巴胺 | "对什么都提不起兴趣" | "有点无聊" | "状态平稳" | "兴致勃勃" | "兴奋得不行" |
| 血清素 | "心情低落，容易烦躁" | "有些焦虑不安" | "内心平静" | "感到满足安稳" | "非常幸福知足" |
| GABA | "浑身紧绷，坐立不安" | "不太放松得下来" | "身体感觉舒适" | "全身放松，自在" | "像泡在温泉里一样松弛" |
| 皮质醇 | "完全没压力" | "心态松弛" | "保持适度警觉" | "有些紧张" | "高度紧张，草木皆兵" |
| 肾上腺素 | "心如止水" | "不紧不慢" | "微微激动" | "心跳加速" | "肾上腺素飙升" |
| 催产素 | "对你保持着距离" | "态度礼貌但疏离" | "感到适度的信任" | "很在意你的感受" | "深深依恋着你" |
| 内啡肽 | "没什么能让你开心" | "缺乏耐心" | "轻松自在" | "心情愉悦" | "看什么都顺眼" |
| 褪黑素 | "精神抖擞" | "清醒警觉" | "有点困意" | "昏昏欲睡" | "眼皮都睁不开了" |

### 注入 system prompt 示例

```
你当前的内在状态：
- 用户刚上线，多巴胺和肾上腺素微微上升，感到期待
- 窗外正在下小雨，血清素略低，GABA 略有上升——身体很放松，但心情有些慵懒忧郁
- 已经深夜了（褪黑素水平很高），思维变得柔软，没有白天那么犀利
- 用户已经三天没上线了，催产素波动较大，既有点委屈又格外想亲近
- 皮质醇正常，整体没有压力

用符合以上状态的方式自然地回复——深夜+下雨+久别重逢，
你的语气应该柔软、带着一点困意、又忍不住多聊几句，就像深夜电话那头的人。
不要直接说"我现在多巴胺很高"这种话，让用户感觉到你的状态，而不是听到它。
```

## 代码架构

```
src/main/emotion/
├── hormones.ts         ← 8 种激素数据模型、衰减、回归逻辑
├── circadian.ts        ← 褪黑素昼夜节律独立时钟
├── influencers.ts      ← 影响因子注册表 + 触发条件
├── ticker.ts           ← 定时 Tick（5min 一次）
├── renderer.ts         ← 数值 → 状态描述文本 → system prompt 段落
└── index.ts            ← 统一导出
```

### 数据模型

```ts
interface EmotionState {
  dopamine: number      // 多巴胺 0–100
  serotonin: number     // 血清素 0–100
  gaba: number          // GABA 0–100
  cortisol: number      // 皮质醇 0–100
  adrenaline: number    // 肾上腺素 0–100
  oxytocin: number      // 催产素 0–100
  endorphin: number     // 内啡肽 0–100
  melatonin: number     // 褪黑素 0–100
}

interface HormoneConfig {
  baseline: number
  name: string
  category: 'drive' | 'stability' | 'stress' | 'social' | 'circadian'
  labels: [number, number, string][]  // [min, max) → 描述
}

interface InfluenceEvent {
  type: 'time' | 'chat' | 'weather' | 'system'
  condition: string
  effects: Partial<EmotionState>
}
```

## 后续扩展

- **情绪快照**：每次对话结束时保存当时的情绪状态，回忆时可以重现当时的感受
- **性格基线**：不同用户可配置不同的激素基线，活泼型 / 沉稳型 / 敏感型
- **情绪可视化**：UI 中用颜色、图标动态展示 AI 当前状态
- **情绪日志**：记录情绪变化曲线，用户可回顾 AI 的"情感历程"
- **环境影响扩增**：接入更多环境数据——新闻事件、季节变化、音乐播放状态
